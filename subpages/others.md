## 音乐工作
{:.lang .lang-zh}

我热衷于古典音乐的欣赏和创作. 我最喜欢浪漫主义和印象主义时期的音乐作品，特别是这些作曲家：拉威尔，德彪西，德沃夏克，王西麟.

- **管弦乐配器/编曲** (2020 ~ )   
  我会少量管弦乐配器和编曲技术. 一些作品可以在[bilibili](https://space.bilibili.com/23222363){:target="_blank"}找到.
- **古典音乐作曲** (2020 ~ )   
  类似地，我会少量古典音乐作曲，特别是基于印象主义色彩和声的室内乐和管弦乐作品. 一些工作可以在[SoundCloud](https://soundcloud.com/h495){:target="_blank"}找到.
- **合唱表演** (2014 ~ )   
  我参加了高中的合唱团和北京理工大学学生合唱团(2018 - 2021). 我的声部是男低音(Bass)或假声男高音(Alto).
- **钢琴演奏** (2005 ~ )   
  我从小学习钢琴，但可惜这种学习并非连续. 
- **长笛演奏** (2019 ~ )   
  我自2019年居家办公/授课时开始学习长笛. 我使用的乐器型号是Yamaha YFL-472.
